## Entenda o básico

Existem várias maneiras de comprar tokens da Conflux Network (CFX). Aqui estão os passos básicos que você 
pode seguir para comprar CFX:

- Abra uma conta em uma exchange que suporte a negociação de CFX. Algumas das exchanges mais populares que 
suportam CFX são Binance, OKEx, Huobi, Gate.io, e MXC.

- Verifique sua conta e faça um depósito na exchange usando uma das opções disponíveis, como transferência 
bancária, cartão de crédito ou débito, ou criptomoedas.

- Pesquise na plataforma da exchange por CFX e escolha a opção de compra.

- Insira a quantidade de CFX que deseja comprar e verifique as taxas e comissões envolvidas na transação.

- Confirme a compra e aguarde até que a transação seja concluída. Os tokens CFX serão creditados em sua 
conta de exchange.



Get started by **creating a new account Lindy**.

Or **try Docusaurus immediately** with **[docusaurus.new](https://docusaurus.new)**.

- [Node.js](https://nodejs.org/en/download/) version 16.14 or above:
  - When installing Node.js, you are recommended to check all checkboxes related to dependencies.

## Criar conta na Lindy Finance

Generate a new Docusaurus site using the **classic template**.

The classic template will automatically be added to your project after you run the command:

```bash
npm init docusaurus@latest my-website classic
```

You can type this command into Command Prompt, Powershell, Terminal, or any other integrated terminal of your code editor.

The command also installs all necessary dependencies you need to run Docusaurus.

## Fazer o cadastro KYC

Run the development server:

```bash
cd my-website
npm run start
```
