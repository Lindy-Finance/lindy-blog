# Conflux - A Rede Blockchain que revoluciona a velocidade das transações

Conflux é uma plataforma blockchain de código aberto que visa fornecer uma infraestrutura de blockchain de alto desempenho e escalabilidade para aplicativos descentralizados (dApps) e contratos inteligentes.

A Conflux é projetada para ser segura, rápida e escalável, com um alto throughput de transações e um 
algoritmo de consenso único chamado Tree-Graph, que combina uma estrutura de árvore e um grafo acíclico 
direcionado (DAG) para melhorar a eficiência e escalabilidade. Além disso, a Conflux implementou um 
mecanismo de ajuste de dificuldade adaptativo (ADT) para garantir que a rede possa se ajustar rapidamente 
a diferentes condições de tráfego de transações.

A Conflux também tem uma governança descentralizada, que é gerenciada por seus detentores de tokens, 
que têm poder de voto para tomar decisões importantes sobre o futuro da rede. A plataforma suporta a 
programação de contratos inteligentes usando a linguagem de programação Solidity, tornando-a compatível 
com a plataforma Ethereum.

A blockchain Conflux tem como objetivo fornecer uma infraestrutura blockchain de alto desempenho, segura e 
escalável para aplicativos descentralizados, bem como soluções para problemas de escalabilidade encontrados 
em outras blockchains, como a Ethereum.
